# Privacy-Preserving Age Verification System

This project demonstrates a prototype system for verifying user age before granting access to social media platforms.

## Features
- Government-style identity verification portal
- UID and fingerprint simulation
- Backend verification using Flask
- Privacy-preserving authentication
- Secure dashboard access

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)

## Architecture
The application separates the social media platform from the identity verification portal. The system verifies age through a backend service and only returns a verification status to the application.
